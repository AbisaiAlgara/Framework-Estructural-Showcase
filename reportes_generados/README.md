# Reportes generados

Esta carpeta contiene ejemplos de salidas automáticas producidas por el framework estructural.

## Tipos de salida

* Memorias técnicas en Markdown.
* Resultados tabulares en CSV.
* Archivos JSON de trazabilidad.
* Revisiones de diseño LRFD.
* Reportes de deflexiones, fuerzas internas, demanda/capacidad y reacciones de base.

## Propósito

Estos archivos muestran la capacidad del framework para generar documentación técnica reproducible, auditable y útil para revisar decisiones de análisis, diseño y optimización estructural.

Los resultados forman parte de un caso aplicado de edificio metálico de cuatro niveles, desarrollado como ejercicio académico y técnico de validación del flujo computacional.
